1. Environment
- nRF5_SDK_15.3.0
- Keil C 5
- USARTHMI for LCD

2. How to build
- Copy project to ...\nRF5_SDK_15.3.0\examples\ble_peripheral\
- Erase and flash softdevice
- Build and flash

3. LCD TJC3224T022_011N
- Run ..\LCD iDe code\USARTHMIsetup.exe to setup tool
- Open file lcd_ide_v1.1.HMI
- Connect LCD to UART and flash
Guide: https://nextion.tech/editor_guide/
